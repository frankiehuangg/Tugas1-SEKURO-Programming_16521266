## Membuat Akun Github
Langkah-langkah membuat akun github:
1. Kunjungi website [Github](https://github.com).
2. Klik tombol **Sign Up** di kanan atas.
3. Masukkan Username, email adress, password, dan verifikasi akun.
4. Klik **Create an Account**.
5. Pilih jenis subscription (Free/Pro) lalu klik **Continue**.
6. Isi data diri yang diminta Github, kemudian klik **Submit**.
7. Verifikasi email lalu login kembali di Github.

## Membuat Project Baru
Langkah-langkah membuat project baru:
1. Klik **Start a project**.
2. Masukkan nama repo, deskripsi repo(optional), jenis repo, dan lainnya.
Nama repository tidak boleh memiliki karakter ' ' dan otomatis akan dikonversikan menjadi '-'. 

## Membuat File Baru
Langkah-langkah membuat file baru:
1. Pada halaman repo, klik **Create new file**.
2. Masukkan isi file.
3. Masukkan pesan commit, kemudian berikan deskripsi untuk penjelasan lebih lanjut.
4. Klik **Commit new file**.
Selain membuat file langsung di repo, kita juga dapat mengupload file pada mesin lokal dengan mengklik **Upload file**

## Mengedit File
Langkah-langkah mengedit file:
1. Pada halaman repo, klik file yang akan diedit isinya.
2. Klik tombol pensil (**Edit this file**).
3. Masukkan perubahan pada file.
4. Masukkan pesan commit, kemudian berikan deskripsi untuk penjelasan lebih lanjut
5. Klik **Commit changes**

## Melihat Perubahan File
Langkah-langkah melihat perubahan yang terjadi pada file:
1. Pada halaman repo, klik file yang akan dilihat perubahannya.
2. Klik **History**.
3. Klik commit yang akan dilihat perubahannya.
Pada commit yang telah dipilih, akan muncul teks yang berisi baris berwarna merah, putih, atau hijau. 
- Baris berwarna merah berarti baris yang dihapus
- Baris Putih berarti baris tidak diubah
- Baris Hijau berarti baris yang ditambahkan

## Melihat Perubahan Repo
Langkah-langkah melihat perubahan yang terjadi pada sebuah repo:
1. Pada halaman repo, klik **commits**
2. KLik commit yang akan dilihat perubahannya.

## Melihat Perubahan Repo Secara Visual
Langkah-langkah melihat perubahan yang terjadi pada sebuah repo:
1. Pada halaman repo, klik **Insights**.
2. Klik **Network**.
