## `git branch`
Didalam sebuah commit, akan terdapat sebuah hash, username, tanggal, dan branch dimana commit berada. Untuk mengetahui branch yang aktif, git mempunyai sebuah pointer yang bernama "head" yang menunjukkan branch sekarang. 

Untuk menampilkan semua branch yang ada pada repo, kita dapat menggunakan command `git branch`

## `git merge`
Untuk menggabungkan 2 atau lebih branch, kita dapat menggunakan command merge. Merge terbagi menjadi 2, yaitu:
- Fast Forward merge terjadi ketika branch yang ingin kita merge berada pada jalur langsung/*direct path*. Kondisi ini terjadi ketika branch yang tergabung merupakan branch langsung branch yang ingin digabung.  
Untuk menggabungkan branch secara fast forward, kita perlu pindah ke branch utama dengan command `git branch master`. Kemudian, kita akan melakukan merge dengan `git merge <nama_branch>`.
![Pindah ke branch master](https://raw.githubusercontent.com/frankiehuangg/Tugas1-SEKURO-Programming_16521266/main/pict/33.png)
![`git merge`](https://raw.githubusercontent.com/frankiehuangg/Tugas1-SEKURO-Programming_16521266/main/pict/34.png)  
Setelah melakukan merge, kita dapat menghapus branch yang sudah tidak kita gunakan. Untuk melakukan penghapusan, dapat digunakan command `git branch -d <nama_branch>`.  
![menghapus branch](https://raw.githubusercontent.com/frankiehuangg/Tugas1-SEKURO-Programming_16521266/main/pict/35.png)

- Three Way merge terjadi ketika branch yang ingin kita merge tidak berada pada jalur yang langsung. Merge ini terjadi dengan membuat sebuah commit baru.
