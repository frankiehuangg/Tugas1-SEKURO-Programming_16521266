## Definisi `.gitignore`
`.gitignore` adalah sebuah file yang berisi kumpulan file yang tidak akan dicommit oleh git. File ini digunakan untuk mencegah file system untuk disimpan oleh git.

## `.gitignore`
Untuk membuat file `.gitignore` pada windows, bisa dilakukan dengan cara biasa membuat file. Untuk menambahkan file yang ingin diabaikan oleh git, cukup ketik nama file tersebut beserta extensinya ke dalam file `.gitignore`

![`git status` sebelum `12.md` ditambahkan di `.gitignore`](https://raw.githubusercontent.com/frankiehuangg/Tugas1-SEKURO-Programming_16521266/main/pict/36.png)
![`git status` setelah `12.md` ditambahkan di `.gitignore`](https://raw.githubusercontent.com/frankiehuangg/Tugas1-SEKURO-Programming_16521266/main/pict/37.png)

Selain mengabaikan sebuah file, `.gitignore` juga dapat digunakan untuk mengabaikan sebuah folder, maupun sekumpulan file dengan ekstensi tertentu. Hal ini bisa dilakukan dengan menambahkan `<nama_folder>/` untuk mengabaikan folder dan `*.<ekstensi>` untuk mengabaikan suatu ekstensi.

`.gitignore` biasanya dilakukan untuk mengabaikan file yang berisi konfigurasi sistem untuk mencegah konfigurasi yang tidak ingin dibagikan kepada orang lain.

Biasanya, Github akan memberikan sekumpulan file yang akan diignore oleh git. File ini dapat diakses di [Github](https://github.com/github/gitignore) atau [gitignore.io](https://www.toptal.com/developers/gitignore)
